# client-side

## Project setup
```
npm install
```

# RUN PWA MODE IN LOCALHOST
```
1. npm run build
2. npm install -g http-server-spa (if you dont have http-server-spa)
3. npm run pwa-test
```

# RUN NON-PWA

npm run dev
```

```
npm run serve

### Compiles and minifies for production
